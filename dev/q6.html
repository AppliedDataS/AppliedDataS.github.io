Let's now consider a neural network with hidden layers.

Training this network is done by back-propagation because we start at the output layer and calculate derivatives backwards towards the input layer with the chain rule.

Let's see how this works.

If we wanted to calculate the derivative of the cost with respect to the weights of the final layer, then this is the same as previously (but now in vector form):

∂ C k ∂ W ( 2 ) = ∂ C k ∂ a ( 2 ) ∂ a ( 2 ) ∂ z ( 2 ) ∂ z ( 2 ) ∂ W ( 2 ) \frac{\partial C_k}{\partial \mathbf{W}^{(2)}} = \frac{\partial C_k}{\partial \mathbf{a}^{(2)}}\frac{\partial \mathbf{a}^{(2)}}{\partial \mathbf{z}^{(2)}}\frac{\partial \mathbf{z}^{(2)}}{\partial \mathbf{W}^{(2)}} ∂W(2)∂Ck​​=∂a(2)∂Ck​​∂z(2)∂a(2)​∂W(2)∂z(2)​

A similar expression can be constructed for the biases.

If we want to calculate the derivative of the cost with respects to weights of the previous layer, we use the expression,

∂ C k ∂ W ( 1 ) = ∂ C k ∂ a ( 2 ) ∂ a ( 2 ) ∂ a ( 1 ) ∂ a ( 1 ) ∂ z ( 1 ) ∂ z ( 1 ) ∂ W ( 1 ) \frac{\partial C_k}{\partial \mathbf{W}^{(1)}} = \frac{\partial C_k}{\partial \mathbf{a}^{(2)}}\frac{\partial \mathbf{a}^{(2)}}{\partial \mathbf{a}^{(1)}}\frac{\partial \mathbf{a}^{(1)}}{\partial \mathbf{z}^{(1)}} \frac{\partial \mathbf{z}^{(1)}}{\partial \mathbf{W}^{(1)}} ∂W(1)∂Ck​​=∂a(2)∂Ck​​∂a(1)∂a(2)​∂z(1)∂a(1)​∂W(1)∂z(1)​

Where ∂ a ( 2 ) ∂ a ( 1 ) \frac{\partial \mathbf{a}^{(2)}}{\partial \mathbf{a}^{(1)}} ∂a(1)∂a(2)​ itself can be expanded to,

∂ a ( 2 ) ∂ a ( 1 ) = ∂ a ( 2 ) ∂ z ( 2 ) ∂ z ( 2 ) ∂ a ( 1 ) \frac{\partial \mathbf{a}^{(2)}}{\partial \mathbf{a}^{(1)}} = \frac{\partial \mathbf{a}^{(2)}}{\partial \mathbf{z}^{(2)}} \frac{\partial \mathbf{z}^{(2)}}{\partial \mathbf{a}^{(1)}} ∂a(1)∂a(2)​=∂z(2)∂a(2)​∂a(1)∂z(2)​

.

This can be generalised to any layer,

∂ C k ∂ W ( i ) = ∂ C k ∂ a ( N ) ∂ a ( N ) ∂ a ( N − 1 ) ∂ a ( N − 1 ) ∂ a ( N − 2 ) … ∂ a ( i + 1 ) ∂ a ( i ) ⏟ from layer  N  to layer  i ∂ a ( i ) ∂ z ( i ) ∂ z ( i ) ∂ W ( i ) \frac{\partial C_k}{\partial \mathbf{W}^{(i)}} = \frac{\partial C_k}{\partial \mathbf{a}^{(N)}} \underbrace{\frac{\partial \mathbf{a}^{(N)}}{\partial \mathbf{a}^{(N-1)}} \frac{\partial \mathbf{a}^{(N-1)}}{\partial \mathbf{a}^{(N-2)}} \ldots \frac{\partial \mathbf{a}^{(i+1)}}{\partial \mathbf{a}^{(i)}} }_{\text{from layer } N \text{ to layer } i} \frac{\partial \mathbf{a}^{(i)}}{\partial \mathbf{z}^{(i)}} \frac{\partial \mathbf{z}^{(i)}}{\partial \mathbf{W}^{(i)}} ∂W(i)∂Ck​​=∂a(N)∂Ck​​from layer N to layer i∂a(N−1)∂a(N)​∂a(N−2)∂a(N−1)​…∂a(i)∂a(i+1)​​​∂z(i)∂a(i)​∂W(i)∂z(i)​
