Data Science dependecies¶

Please note that the following statement is used to import the libraries used in the demo.

 
    
    
    
    

  
   install matplotlib
   install graphviz
   install networkx
   install numpy

Alternatively you can use the file requirements.txt and use pip to install all the libraries in the text file

 install -r requirements.txt >log.txt

If you're on macOS X you might need other preperations
